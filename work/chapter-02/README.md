# Chapter 02 Workshop — Toolchain Doctor

Build a deterministic toolchain diagnostics CLI.

Commands: `check`, `explain <check>`, `help`, `demo`, `reset`, `stats`, `verify`, `quit`, then persistence/replay at Level 5.

Run:

```bash
odin run work/chapter-02
odin test work/chapter-02
```

Use an injected environment adapter in tests; do not depend on the real machine.
