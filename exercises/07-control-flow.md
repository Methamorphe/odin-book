# Chapter 7 Exercises — Control Flow

[← Chapter](../book/07-control-flow.md) · [Solutions →](../solutions/07-control-flow.md)

1. Classify a temperature with `if/else if/else`.
2. Rewrite three nested validations as guard clauses.
3. Build a command router with `switch` and a default case.
4. Use a condition switch to classify health ranges.
5. Sum only positive values using `continue`.
6. Find a target in a two-dimensional array using a labeled break.
7. Remove negative values from a slice using a write cursor.
8. Describe the invariant of a loop that computes a running maximum.
9. Compare the cost of a nested duplicate search with a map-based solution.
10. Challenge: implement a small enum-driven connection state machine.
