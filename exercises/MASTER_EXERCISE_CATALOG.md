# Exercise Catalog — 240 Interactive Exercises

The original prompt-only catalog has been replaced by the [Interactive Exercise Workbook](INTERACTIVE_WORKBOOK.md).

The workbook contains five cumulative levels for each of the 48 chapters:

1. **Guided build** — implement one useful command and observable result.
2. **Interactive shell** — add a command loop, help, demo, reset, stats and quit.
3. **Test-first boundaries** — add reproducible automated tests for malformed and boundary input.
4. **Instrument and explain** — collect chapter-specific metrics and compare constrained runs.
5. **Persistent product** — add save, load, replay and invariant verification.

That gives **240 practical exercises**. Each chapter builds one working program rather than a collection of disconnected answers.
