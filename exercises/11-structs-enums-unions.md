# Chapter 11 Exercises — Structs, Enums, and Unions

[← Chapter](../book/11-structs-enums-unions.md) · [Solutions](../solutions/11-structs-enums-unions.md)

1. Define a `Rectangle` struct and procedures for area and perimeter.
2. Create a `Traffic_Light` enum and a procedure returning the next state.
3. Model file permissions with an enum and a bit set.
4. Replace a struct containing `is_loading`, `is_ready`, `has_failed`, `progress`, `resource_id`, and `error_message` with a union-based state model.
5. Inspect the size and alignment of two structs containing the same fields in different orders. Explain the difference.
6. Build an `Event` type with common metadata and three variant payloads. Print each event using a type switch.
