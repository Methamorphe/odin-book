# Chapter 12 Exercises — Pointers and Addressability

[← Chapter](../book/12-pointers-addressability.md) · [Solutions](../solutions/12-pointers-addressability.md)

1. Write `swap` for two integers using pointers.
2. Define a `Transform` and mutate it through a pointer parameter.
3. Explain why returning a pointer to a local variable is invalid.
4. Create a dynamic array, take a pointer to an element, then identify every operation that could invalidate it.
5. Rewrite a pointer-plus-count procedure as a slice-based Odin API.
6. Design a generation-based handle for a pool slot and describe its validation rules.
